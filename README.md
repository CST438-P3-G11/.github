# CST438 Project 3 Group 11 - PlaneSpOtter

## Our Team

Neil Cabanilla, Kristopher Church, Daniel Everman, James Fisher

## About Our Project

PlaneSpOtter is a web app where users can compete against one another to see who knows their planes the best. Players are presented with a picture of a plane and a set of potential answers which the user has to guess from. 
Stats for right/wrong answers are tracked in the database and can be viewed on a leaderboard of all the players.

## Tech Stack

### Frontend

- Uses React Native, hosted on Vercel
- Mobile version was built with Android in mind

### Backend

- API Done with Spring Boot and hosted on Render
- Database hosted on Supabase, using PostgreSQL
